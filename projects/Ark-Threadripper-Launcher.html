What is Ark?

Ark: Survival Evolved (stylized as ARK) is a 2015 action-adventure survival video game developed by Studio Wildcard. In the game, players must survive being stranded on one of several maps filled with roaming dinosaurs, fictional fantasy monsters, and other prehistoric animals, natural hazards, and potentially hostile human players.

I was a founder of one of it's servers called Ark: Threadripper, containing many 'refugees' from the official dedicated servers from Studio Wildcard, I've spent many thousands of hours in this game and I do not regret a single second of it. I have made many friends and learned allot along the way. It has been a great experience and I would recommend it to anyone who enjoys survival games. Now since I had programming knowledge I thought I could make logging into the servers allot easier with a dedicated launcher so I sat down and made one using Visual Studio.

Features

Every time I added a feature or some fixes the release version was incremented.

  • Server selection: On the left panel you can select which server you want to connect to. Each server has its own map with different creatures and player inhabitants. Once "Play" is clicked the launcher will start the game with the selected server.

  • Styling: You can change the appearance of the whole launcher via changing the server selection

  • Mod button: View all the mods from the server (not displayed here)

  • Clear cache: Clears out your game cache which helps stop certain crashes and errors

  • Auto update: The launcher will check for updates and download them automatically
